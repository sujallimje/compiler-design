# Flex Program to count the number of lines and characters

Open cmd in directory

### Flex to C

```
flex code.l
```

###Compile C

```
gcc lex.yy.c
```

## Run output file

```
a.exe file.txt
```

> [!TIP]
> Fell free to tweak file.txt to change inputs.

> [!NOTE]
> Install flex , bison and gcc compiler
