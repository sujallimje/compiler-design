# Flex program to check if it is a well formed parenthesis or not



Open cmd in directory

### Flex to C
```
flex parenthese.l
```

###Compile C
```
gcc lex.yy.c
```
## Run output file
```
a.exe file.txt
```

> [!TIP]
> Fell free to tweak file.txt to change inputs. 


> [!NOTE]
> Install flex , bison and gcc compiler
